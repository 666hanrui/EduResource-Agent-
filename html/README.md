# Archived static experiments

This directory is not part of the active React application.

The active product entry is:

- `frontend/index.html`
- `frontend/src/main.tsx`

The files in this directory were one-off visual exploration drafts created during design iteration. They are intentionally kept outside the formal frontend route tree and should not be treated as current product pages.

Current formal surfaces:

- Landing / role entry: `frontend/src/components/ProjectLanding/`
- Student app: `frontend/src/App.tsx`
- Teacher portal: `frontend/src/components/TeacherPortal/`

If a static experiment is needed again, move it into `docs/design-experiments/` or recreate it from the React implementation rather than wiring it into production routes.
